## PORT - 같은 IP 내에서 프로세스 구분

8090포트 - 게임서버 연결
21000포트 - 화상통화 연결
10010포트 - 웹브라우저 요청

등등

- IP가 아파트면 PORT가 몇동 몇호!

## PORT

- 0 ~ 65535 할당가능
- 0 ~ 1023: 잘 알려진 포트, 사용하지 않는 것이 좋음

    - FTP - 20, 21
    - TELNET - 23
    - HTTP - 80
    - HTTPS - 443
